# 📦 自动点击器 APK - 用户指南

## 🎯 项目说明

这是一个使用 Flutter 开发的独立 Android 应用，可以自动点击微信小程序中的按钮和选择框。

### 核心功能
- ✅ **坐标点击**：精确点击屏幕指定位置
- ✅ **文本点击**：根据按钮文字自动查找并点击
- ✅ **动作序列**：支持多个点击动作按顺序执行
- ✅ **延迟控制**：每个动作之间可设置延迟
- ✅ **独立运行**：无需电脑，手机上直接使用

## 🚀 如何获取 APK

### 当前状态

由于系统限制（缺少 Android SDK），无法在当前环境直接构建 APK。

### 获取方式

#### 方式 1：在有 Android SDK 的机器上构建（推荐）

**前提条件：**
- 已安装 Java JDK
- 已安装 Android SDK
- 已安装 Flutter（已有）

**构建步骤：**

1. **安装 JDK**
   ```bash
   sudo apt install openjdk-17-jdk
   ```

2. **安装 Android SDK**
   - 下载 Android Studio 或使用 commandlinetools
   - 详细步骤见 `BUILD_GUIDE.md`

3. **配置环境变量**
   ```bash
   export ANDROID_HOME=~/Android/sdk
   export PATH=$PATH:$ANDROID_HOME/platform-tools
   source ~/.bashrc
   ```

4. **验证环境**
   ```bash
   flutter doctor -v
   ```

5. **构建 APK**
   ```bash
   cd ~/.openclaw/workspace/auto_clicker_app
   flutter build apk --release
   ```

6. **获取 APK**
   ```
   build/app/outputs/flutter-apk/app-release.apk
   ```

#### 方式 2：使用在线构建服务

- GitHub Actions
- Codemagic
- AppCenter

详见 `BUILD_GUIDE.md`

## 📱 安装和使用

### 安装步骤

1. **获取 APK 文件**
   - 按照上述方式构建或下载

2. **安装到手机**
   ```bash
   # 方法 1：ADB 安装
   adb install -r app-release.apk

   # 方法 2：直接在手机上安装
   # 将 APK 传输到手机，点击安装
   ```

3. **启用无障碍服务** ⚠️ 必须步骤
   ```
   设置 → 辅助功能 → 无障碍 → 自动点击器 → 开启
   ```
   或在 App 内点击右上角设置图标

### 使用方法

#### 添加点击动作

1. 点击 **"添加点击动作"** 按钮

2. 选择点击类型：
   - **坐标点击**：输入 X、Y 坐标
   - **文本点击**：输入按钮上的文字（推荐）

3. 设置延迟时间（毫秒，默认 500ms）

4. 点击 **"添加"**

#### 执行自动点击

1. 添加完所有动作后
2. 打开微信小程序到目标界面
3. 点击 **"开始"** 按钮
4. App 会按顺序自动执行

#### 停止自动点击

点击 **"停止"** 按钮即可

## 💡 使用技巧

### 如何获取屏幕坐标？

**方法 1：开发者选项**
```
设置 → 辅助功能 → 开发者选项 → 指针位置（开启）
```
开启后触摸屏幕会显示坐标。

**方法 2：估算坐标**
- 截图后使用图片工具查看
- 一般手机屏幕：宽 1080-1440，高 1920-2400

**方法 3：文本点击（推荐）**
- 无需知道精确坐标
- 只需输入按钮文字
- 例如："登录"、"确认"、"下一步"

### 测试微信小程序

1. 在微信小程序中找到要点击的按钮
2. 记录按钮文字
3. 在 App 中添加文本点击动作
4. 打开小程序，点击"开始"测试

### 建议的测试流程

```
1. 先添加 1 个点击动作
2. 测试是否能正确点击
3. 确认无误后，添加更多动作
4. 逐步完善点击序列
```

## ⚠️ 注意事项

### 必须满足的条件

✅ **无障碍服务已开启**
✅ **手机屏幕已解锁**
✅ **微信小程序在前台显示**
✅ **按钮文字或坐标正确**

### 常见问题

#### 问题：无法点击

**可能原因：**
- 无障碍服务未开启
- 微信不在前台
- 坐标或文字错误

**解决方法：**
1. 检查无障碍服务是否开启
2. 确保小程序界面显示在屏幕上
3. 尝试使用文本点击替代坐标点击
4. 增加延迟时间

#### 问题：点击位置不准确

**解决方法：**
1. 开启"指针位置"获取精确坐标
2. 使用文本点击（更可靠）
3. 调整点击坐标

#### 问题：无障碍服务自动关闭

**解决方法：**
```
设置 → 应用管理 → 自动点击器 → 电池优化 → 不优化
设置 → 应用管理 → 自动点击器 → 权限管理 → 后弹窗 → 允许
```
