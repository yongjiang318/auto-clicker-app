# 🤖 自动点击器 APK

这是一个独立的 Android 应用，可以自动点击微信小程序中的按钮和选择框。

## 📋 功能特性

- ✅ 坐标点击：指定屏幕坐标进行自动点击
- ✅ 文本点击：根据文本内容查找并点击
- ✅ 动作序列：支持多个点击动作按顺序执行
- ✅ 延迟控制：每个动作之间可设置延迟时间
- ✅ 独立运行：无需电脑，手机上即可使用

## 🚀 安装步骤

### 1. 启用无障碍服务

**首次使用必须启用无障碍服务：**

```
设置 → 辅助功能 → 无障碍 → 自动点击器 → 开启
```

或直接在 App 内点击右上角的设置图标。

### 2. 安装 APK

将 `app-release.apk` 文件传输到手机，直接安装即可。

## 📖 使用方法

### 添加点击动作

1. 点击 **"添加点击动作"** 按钮
2. 选择点击类型：
   - **坐标点击**：输入 X 和 Y 坐标
   - **文本点击**：输入要点击的按钮文字
3. 设置延迟时间（毫秒）
4. 点击 **"添加"**

### 执行自动点击

1. 添加完所有点击动作后
2. 打开微信小程序到目标界面
3. 点击 **"开始"** 按钮
4. App 会按照顺序自动执行所有点击动作

### 停止自动点击

点击 **"停止"** 按钮即可暂停。

## 💡 使用技巧

### 如何获取屏幕坐标？

**方法 1：使用开发者选项**
```
设置 → 辅助功能 → 开发者选项 → 指针位置（开启）
```
开启后，手指触摸屏幕会显示坐标。

**方法 2：使用截图工具**
1. 截取屏幕
2. 使用图片编辑工具查看坐标
3. 估算按钮中心位置

**方法 3：文本点击（推荐）**
- 使用文本点击无需知道精确坐标
- 只需输入按钮上的文字即可
- 更简单，更可靠

### 测试微信小程序

1. 先在微信小程序中找到要点击的按钮
2. 记录按钮文字（例如："登录"、"确认"）
3. 在 App 中添加文本点击动作，输入该文字
4. 打开小程序，点击"开始"

### 注意事项

⚠️ **重要提示：**
- 确保无障碍服务已开启
- 确保手机屏幕保持解锁
- 确保小程序界面已显示在屏幕上
- 不同微信版本界面可能不同，需相应调整
- 建议先测试单个动作，确认无误后再添加多个

## 🔧 故障排除

### 问题：无法点击微信小程序

**解决方案：**
1. 确认无障碍服务已开启
2. 检查微信是否在后台（无障碍配置中已设置只监听微信）
3. 尝试使用文本点击替代坐标点击

### 问题：点击位置不准确

**解决方案：**
1. 开启"指针位置"选项获取精确坐标
2. 增加点击延迟时间
3. 尝试使用文本点击

### 问题：无障碍服务自动关闭

**原因：**
某些手机（华为/小米等）会自动优化后台服务

**解决方案：**
```
设置 → 应用管理 → 自动点击器 → 电池优化 → 不优化
设置 → 应用管理 → 自动点击器 → 权限管理 → 后弹窗 → 允许
```
